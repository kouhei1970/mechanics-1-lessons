# 基礎力学Ⅰ — 触って学べるインタラクティブ授業資料

教科書の内容を「読むだけ」でなく、**自分で値を動かして確かめられる**教材へ。
力の大きさ・角度のスライダー、図のドラッグ、採点つき練習を、ブラウザだけで。各ページに音声解説つき。

🔗 **公開ページ:** https://kouhei1970.github.io/mechanics-1-lessons/

---

## 使い方

- 各教材を開き、**→／Space** でページ送り（**←** で戻る）。
- 右上の **🔊 解説** でそのページの音声解説を再生（edge-tts による合成音声）。
- 右下の **⛶** でプロジェクタ向けの全画面表示。
- 練習問題は数値を入力して **採点**、**答えを見る**、**やり直す** が使えます。

## 構成

```
index.html                   教材一覧（このページ）
lessons/lessonNN/index.html  各回のデッキ（+ audio/ 音声, narration.json, tts_dictionary.py）
```
