解説

前回の復習

並進運動の基礎

運動の記述（速度は位置の時間微分、加速度は速度の時間微分）：

v = ds/dt    v = dr/dt    a = dv/dt = d²r/dt²

行程（経路の全長・スカラー）と変位（ベクトル）を区別する。

等加速度運動・自由落下：

v = a₀t + v₀    s = ½a₀t² + v₀t + s₀    v² − v₀² = 2as

自由落下は a = g = 9.80 m/s² の特別な場合。前回はすべて直線運動。今日はこれを2次元へ拡張する。

解説

一般の運動（2次元）

2次元の位置ベクトル

質点の位置を時間の関数として表す：

r(t) = x(t) ex + y(t) ey
  • x(t)：x 方向の座標成分
  • y(t)：y 方向の座標成分
  • ex, ey：単位ベクトル

運動は各成分ごとに独立に解析できる。

解説

一般の運動（2次元）

速度ベクトルと速さ

位置ベクトルの時間微分：

v(t) = dr/dt = (dx/dt) ex + (dy/dt) ey = vx ex + vy ey

速さ（速度の大きさ）：

| v | = √( vx² + vy² )

速度の向き（経路の接線方向）：

tanα = vy / vx

速度ベクトルは、いつでも経路に接する向きを指す。これが「接線方向」の意味。

解説

一般の運動（2次元）

加速度ベクトル

速度ベクトルの時間微分：

a(t) = dv/dt = d²r/dt² = ax ex + ay ey
ポイント
加速度は速度の変化を表す。
速さの変化だけでなく、向きの変化も加速度に含まれる。

→ 速さが一定の等速円運動でも、向きが変わり続けるので、加速度はゼロではない！

解説

接線加速度と法線加速度

加速度の分解

曲線運動では、加速度を2つの成分に分解すると便利：

接線加速度 at
  • 経路の接線方向
  • 速さを変える
at = dv/dt
式 (5.22)
法線加速度 an
  • 経路の法線方向（内向き）
  • 向きを変える
an = v² / ρ
式 (5.23)

全加速度の大きさ： a = √( at² + an² )   式 (5.24)

解説

接線加速度と法線加速度

接線加速度の意味

at = dv/dt = d| v |/dt
  • at > 0：質点は加速（速さが増える）
  • at < 0：質点は減速（速さが減る）
  • at = 0：速さ一定（等速運動）
例：等速円運動
速さが一定 → at = 0、しかし向きは変化 → an ≠ 0
解説

接線加速度と法線加速度

法線加速度と曲率半径

an = v² / ρ
  • ρ（ロー）：曲率半径（経路の曲がり具合、教科書では r と表記）
  • an はつねに曲率中心を向く（内向き）
  • ρ → ∞（直線）ならば an → 0

経路 y = f(x) の曲率半径：

ρ = [ 1 + (dy/dx)² ]3/2 / | d²y/dx² |
  • 円運動：ρ = r（一定）
  • 曲がりが急なほど ρ は小さい
注意　法線加速度は速さを変えない。向きのみを変化させる。

確認

接線・法線加速度を分解してみる

放物運動の経路（青）の上で、赤い点（質点）をドラッグすると、その点での加速度（下向き重力 g）が、緑の接線方向と橙の法線方向に分かれて表示されます。

経路 加速度 g（下向き） 接線成分 at 法線成分 an

上り坂では at が負（減速）、最高点で at = 0、下り坂では正（加速）。法線成分はつねに経路の内側（下向き）を向く。

接線加速度と法線加速度

接線・法線分解のまとめ

a = at et + an en

解説

放物運動

放物運動とは

放物運動：重力のみが作用する2次元運動。

初期条件（t = 0）：

  • 初速度の大きさ：v₀
  • 投射角：θ（水平面からの角度）
  • 初期位置：原点 (0, 0)

初速度の成分：

vx0 = v₀ cosθ,  vy0 = v₀ sinθ
解説

放物運動

放物運動の運動方程式

空気抵抗を無視すると、各成分の加速度は：

ax = 0  (5.25),   ay = −g  (5.26)
水平方向（等速運動）
vx(t) = v₀ cosθ (5.27)
x(t) = (v₀ cosθ) t (5.29)
鉛直方向（等加速度運動）
vy(t) = v₀ sinθ − gt (5.28)
y(t) = −½gt² + (v₀ sinθ) t (5.30)

水平は速さ一定、鉛直は自由落下と同じ。二つの直線運動の重ね合わせ。

解説

放物運動

軌道の方程式

x(t) から時間 t を消去：  t = x / (v₀ cosθ)

y(t) に代入すると：

y = − g / (2v₀² cos²θ) · x² + (tanθ) x   式 (5.31)
  • x の2次関数 → 放物線（parabola）
  • 名前の由来：放物運動 ＝ 放物線を描く運動

確認

初速 v₀ と投射角 θ で軌道が変わる

軌道 最高点 H 到達距離 R

θ を 45° に合わせると、同じ初速で最も遠くまで飛びます。スライダーを動かして確かめましょう。

解説

放物運動

最高点の高さ

最高点では鉛直方向の速度がゼロ：vy = 0

v₀ sinθ − g t₁ = 0   ⟹   t₁ = v₀ sinθ / g  (5.32)

t₁ を y(t) に代入：

ymax = (v₀ sinθ)² / (2g)  式 (5.33)
  • θ = 90°（真上）のとき ymax は最大（鉛直投げ上げ）
  • ymax は v₀² に比例
解説

放物運動

水平到達距離

着地条件：y = 0（t > 0）

v₀ sinθ · t − ½g t² = 0   ⟹   t₂ = 2v₀ sinθ / g  (5.34)

t₂ を x(t) に代入：

xmax = v₀² sin2θ / g  式 (5.35)
  • sin2θ が最大 → 2θ = 90° → θ = 45°
  • 飛行時間：t₂ = 2t₁（対称性）
解説

最適角度

最大飛距離と最適角度

R = v₀² sin2θ / g において sin2θ = 1 のとき最大：

θ = 45° ⟹ Rmax = v₀² / g

右のグラフは到達距離 R を投射角 θ の関数として描いたもの。45° で頂点（最大）になる。

操作

確認

補角で飛距離が等しいことを確かめる

sin2θ = sin2(90°−θ) より：

投射角 θ と 90°−θ の飛距離は等しい
30°
角度 θ 補角 90°−θ

θ を 30° にすると補角は 60°。2本の軌道は形がちがっても、到達距離 R は同じ。低い方が飛行時間も短い。

操作

到達限界

到達限界（放物運動の包絡線）

すべての投射角 θ の軌道の包絡線は放物線：

y₀ = v₀²/(2g) − g/(2v₀²) x₀²  式 (5.38)
45°

赤い太線が包絡線。どの軌道もこの線の内側にしか届きません。包絡線の内側が到達可能な領域で、最大飛距離は θ = 45° で v₀²/g です。

例題

例題12.1：放物運動の基本

問題

初速 v₀ = 20 m/s、投射角 θ = 30° で水平面から物体を投げた。空気抵抗は無視する。（g = 9.80 m/s²）

  • ① 最高点の高さ H を求めよ。
  • ② 水平到達距離 R を求めよ。
  • ③ 飛行時間 t₂ を求めよ。

次のページで解いてみましょう。まずは自分で式を立ててみてください。

解説

例題

例題12.1：解答

v₀ = 20 m/s、θ = 30°、g = 9.80 m/s²

(1) 最高点の高さ： H = v₀² sin²θ / 2g = (20² × 0.25) / 19.60 = 400×0.25/19.60 ≈ 5.10 m
(2) 水平到達距離： R = v₀² sin2θ / g = (400 × sin60°)/9.80 = 400×0.866/9.80 ≈ 35.3 m
(3) 飛行時間： t₂ = 2v₀ sinθ / g = (2×20×0.5)/9.80 ≈ 2.04 s

sin30° = 0.5、sin60° = 0.866 を使う。最高点が約 5 m、約 35 m 先に、約 2 秒で着地する。

例題

例題12.2：最高点の接線・法線加速度

問題　放物運動の最高点における接線加速度と法線加速度を求めよ。
解答：最高点では vy = 0、vx = v₀ cosθ
  • 速度は水平方向のみ → 接線方向 ＝ 水平方向
  • 重力 g は鉛直下向き → 接線方向に対して垂直
at = 0,   an = g
最高点での曲率半径：an = v²/ρ = g ⟹ ρ = v₀² cos²θ / g
操作

教科書 例題5.7（p.90）

落下ボールへの投石

高さ h のビル屋上からボールを自由落下させると同時に、水平距離 s の地点から石を仰角 θ で投げる。石がボールに命中する仰角は？

tanθ = h / s

重力は石とボールを同じだけ引き下げる。重力を除くと石はまっすぐ飛び、ボールは屋上に留まる。だから石は初めボールがあった点を狙えば必ず当たる。相対運動は直線的になる。

まとめ

本日のまとめ

本日のまとめ

  • ① 2次元の運動：r(t), v(t), a(t) を成分で表す
  • ② 接線・法線分解：at = dv/dt（速さを変える）、an = v²/ρ（向きを変える）
  • ③ 放物運動：水平＝等速運動、鉛直＝等加速度運動（a = −g）、軌道は放物線
  • ④ 重要公式：ymax = (v₀ sinθ)²/2g（5.33）、xmax = v₀² sin2θ/g（5.35）、最適角 45°
  • ⑤ 到達限界（包絡線）：y₀ = v₀²/2g − g/(2v₀²) x₀²（5.38）

速さは接線が担当、向きは法線が担当。放物運動は二つの直線運動の重ね合わせ。
