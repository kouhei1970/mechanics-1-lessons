前回の復習

一般の運動・放物運動

2次元の運動は、位置を r = x ex + y ey と書き、各成分を独立に解析できる。

v = dr/dt,　a = dv/dt

加速度は、速さを変える接線加速度 at と、向きを変える法線加速度 an = v2/ρ に分けられる。

放物運動（§5.1.7）

水平は等速、鉛直は等加速度（a = −g）。

xmax = v02 sin 2θ / g

今回はこの続きとして、向きを変える運動の代表である円運動を、角度の言葉で本格的に扱います。

図解説

回転運動の記述（§5.2）

回転運動とは

  • 物体が点 O を通る軸まわりに回る運動
  • 異なる2点 A, B の接線速度は vA = rAω, vB = rBω で等しくない
  • 並進運動のように1点で代表できない
  • 角度の言葉で記述する：角変位 θ［rad］、角速度 ω、角加速度 α
同じ角速度でも、外側ほど速い
解説

回転運動の記述（§5.2）

角速度・角加速度

角速度 ω

ω = dθ/dt　［rad/s］

反時計回りを正とする。1秒あたり何ラジアン回るか。

角加速度 α

α = dω/dt = d²θ/dt²　［rad/s²］

直線運動の加速度 a に対応。

回転数 n［rpm］との関係（式 5.39）
ω = 2πn / 60
n：毎分回転数

例： n = 3000 rpm のとき
ω = 2π × 3000 / 60 = 100π ≈ 314 rad/s

等速円運動（§5.3）

等速円運動：周期と振動数

条件：角速度 ω = 一定 ⇒ 角加速度 α = 0、接線加速度 at = rα = 0。向心加速度 an = rω² = v²/r のみ（常に中心向き）。

周期 T（1回転の時間）

T = 2π / ω　［s］

振動数 f（毎秒の回転数）

f = 1/T = ω/2π　［Hz］
相互関係：　ω = 2πf = 2π/T
図解説

等速円運動（§5.3）

等速円運動の向心加速度

速度（青）は向きを変え続け、加速度（赤）は常に中心向き
|v| = rω = 一定
|an| = rω² = v²/r

速さは一定でも、向きが変わるから加速度はゼロでない。

an は常に中心を向く。これが円運動を維持する加速度＝向心加速度。

解説

等角加速度円運動（§5.3）

等角加速度円運動

角加速度 α が一定のとき、直線運動の等加速度運動と完全に同じ形の式が成り立つ：

ω = ω0 + αt　（式 5.51）
θ = ω0t + ½ αt²　（式 5.52）
ω² − ω0² = 2αθ　（式 5.53）

ω0：初角速度、α：角加速度（一定）、θ：角変位。直線運動の v = v0 + at ほかと、x→θ・v→ω・a→α と読みかえれば同じ。

解説

例題 13.1

等角加速度回転

問題：静止から一定の角加速度 α = 3.0 rad/s² で回り始めた半径 r = 0.20 m の円板。t = 4.0 s 後の (a) 角速度、(b) 縁の速さ、(c) 回転数 を求めよ。
(a) ω = ω0 + αt = 0 + 3.0 × 4.0 = 12 rad/s
(b) v = rω = 0.20 × 12 = 2.4 m/s
(c) θ = ½αt² = ½ × 3.0 × 4.0² = 24 rad、回転数 = θ/2π = 24/2π ≈ 3.8 回転
解説

例題 13.2

回転機械の減速

問題：ω0 = 300 rad/s で回るモータが一定の角加速度で減速し、t = 10 s で停止した。(a) 角加速度、(b) 停止までの回転数 を求めよ。
(a) ω = ω0 + αt より　0 = 300 + α × 10　⇒　α = −30 rad/s²（減速）
(b) θ = ω0t + ½αt² = 300 × 10 + ½ × (−30) × 10² = 1500 rad
　　回転数 = 1500 / 2π ≈ 239 回転

角加速度が負になるのが「減速」。符号の意味は直線運動の負の加速度と同じ。

例題 13.3

2段歯車列

問題：歯車A（NA=80）が ωA=5 rad/s で回転。歯車B（NB=40）がAとかみ合い、歯車C（NC=10, Bと同軸）が歯車D（ND=30）とかみ合う。Dの角速度を求めよ。

相対運動（§5.4）

相対運動とは

基本的な考え方：止まっている点を基準に見た運動を絶対運動、動いている点を基準に見た運動を相対運動という。運動は「どの観測者から見るか」で異なる。
  • 2点 P, Q の絶対速度を vP, vQ とする
  • 速度の差 vQ − vP が相対速度

日常の例

自分が車に乗って隣の車を見ると、地面から見るより遅く見える。観測者の速度を引いた分が「見かけの速度」。

図解説

相対運動（§5.4, 図5.21）

相対速度の公式

vQ/P = vQ − vP,　aQ/P = aQ − aP

「Pから見たQの速度」＝「Qの速度」−「Pの速度」

  • Pを止めるには −vP を両者に加える
  • するとQの速度は vQ − vP に見える
Pから速度を引くイメージ

相対運動（§5.4）

2次元の相対運動：解き方

方法1：ベクトル三角形
三角形を描き、余弦定理・正弦定理で大きさと角度を出す。

vQ/P² = vQ² + vP² − 2vQvP cos β

方法2：成分分解
x成分・y成分に分けて、それぞれ引き算する。

vQ/P,x = vQx − vPx
vQ/P,y = vQy − vPy

どちらでも解ける。直交する2速度なら成分分解＋三平方の定理が速い。

例題 13.4

雨と歩行者

問題：雨が鉛直下方に vR=5.0 m/s で降る。人が水平に vP=3.0 m/s で歩くとき、人から見た雨の速さと方向を求めよ。
vR/P = vR − vP
vR/P = √(vR² + vP²) = √(5.0² + 3.0²) = √34 ≈ 5.8 m/s
tan θ = vP/vR = 3.0/5.0,　θ ≈ 31°（鉛直から前方へ）
人から見ると雨は斜め前から

例題 13.5

等速円運動：周期と向心加速度

問題：長さ r = 0.50 m のひもの先につけた小球を、水平面内で一定の速さで回す。1回転に T = 0.40 s かかった。(a) 角速度 ω、(b) 球の速さ v、(c) 向心加速度 an を求めよ。
(a) ω = 2π/T = 2π/0.40 ≈ 15.7 rad/s　（周期から角速度へ）
(b) v = rω = 0.50 × 15.7 ≈ 7.9 m/s　（橋渡しの式 v = rω）
(c) an = rω² = 0.50 × 15.7² ≈ 123 m/s²　（= v²/r でも同じ。ただし丸め前の v = 7.85 m/s で計算すると 123 m/s² に一致。丸めた 7.9 を使うと 124.8 と僅かにずれる）

向心加速度は約 123 m/s²、重力加速度 g ≈ 9.8 m/s² の約13倍。半径が小さく速さが大きいほど大きくなる。an = rω² と v²/r は同じ値（ω = v/r を代入すれば一致）。

解説

つまずきポイント

「遠心力で外へ飛ぶ」は本当か？

よくある誤解：円運動する物体には「外向きの遠心力」がはたらいて、外へ引っぱられている。
  • 実際の加速度は中心向き。等速円運動の加速度 an = v²/r はつねに中心を向く。だから必要な力（向心力 F = man = mv²/r）も中心向きで、外向きの力ではない。
  • ひもが切れたら？向心力が消えるので、物体は中心から離れる方向ではなく、その瞬間の接線方向（速度の向き）へ直進する（慣性の法則）。外へ放り出されるように見えるのは、まっすぐ進もうとするから。
  • 「遠心力」を感じる理由：自分が回転する乗り物の中（＝加速する観測者）にいると、見かけの外向きの力を感じる。これは回転系でだけ現れる見かけの力で、地面に立つ観測者から見ると存在しない。

ポイント：地面（静止系）で考えるなら、円運動を保つのは中心向きの向心力だけ。カーブで体が外へ押される感覚は、まっすぐ進もうとする慣性の現れ。
