解説

前回の復習：摩擦 → 運動学へ

前回まで：静力学

  • 力の釣合い・物体が静止する条件を扱った
  • すべり摩擦 fs = μsR や転がり摩擦は、力の原因に着目した話題

今回から：運動学（kinematics）

  • 力の原因を問わず、位置 x(t)・速度 v(t)・加速度 a(t) の時間変化に着目
  • 物体を1質点で代表できる運動を並進運動という（教科書§5.1, p.78）
この回のキーワード：　位置 x(t)　　速度 v(t)　　加速度 a(t)　（すべて時間 t の関数）
解説図

§5.1.1

直線運動と曲線運動（§5.1.1）

  • 直線運動（linear motion）：経路が一直線
  • 1次元の座標 x ひとつで記述
  • 例：レール上の台車
  • 曲線運動（curved motion）：経路が曲線
  • 2次元以上の座標で記述
  • 例：投げた球の運動
上：一直線の経路／下：曲がった経路

本講義ではまず直線運動を扱う（教科書§5.1.1）。

解説

§5.1.2

行程・距離・変位（§5.1.2）

用語種類性質
行程（course length）s スカラー量 経路に沿った全長／常に s ≥ 0／往復しても加算
変位（displacement）Δr ベクトル量 始点 P から終点 Q／Δr = rQ − rP／経路によらない
距離（distance）|Δr| ベクトルの大きさ 変位の大きさ |Δr|／始点と終点の直線距離／常に ≥ 0
用語の関係（混同しやすい）： 教科書では「距離」＝変位ベクトルの大きさ |Δr|（始点・終点間の直線長）、「行程」＝経路に沿った長さ。一般に |Δr| ≤ s。
操作

確認

変位ベクトルを動かして確かめる

始点 P を原点に固定、終点 Q を動かせます。大きさ Δr = √(Δx² + Δy²)（式5.1, ＝距離）、向き α = arctan(Δy/Δx)（式5.2）。

終点 Q をドラッグすると、成分 Δx・Δy、大きさ Δr（＝距離）、向き α が変わる。

解説

§5.1.3

速さと速度（§5.1.3）

平均の速さ（average speed, スカラー）： v = Δs / Δt　（行程 ÷ 経過時間）
速さ（瞬間, scalar）： v = limΔt→0 Δs/Δt = ds/dt　（式5.4）
速度（velocity, vector）： v = limΔt→0 Δr/Δt = dr/dt　（式5.5）

速さはスカラー（大きさのみ・式5.4）、速度は大きさと向きをもつベクトル（式5.5）。速度の大きさが速さ。

日常語では同じだが、物理では速さ＝スカラー、速度＝ベクトルと区別する。

解説

直線運動の速度＝位置の時間微分

直線運動では位置を x(t) という時間の関数で表すと、速度はその時間微分になります。

v(t) = dx / dt
  • v(t) の符号は運動の向きを表す（＋：正の向き、−：逆向き）
  • 速さは速度の大きさ |v(t)|（常に ≥ 0）
  • 単位は m/s（位置 m を時間 s で微分）
微分の練習　x(t) = 3t² − 2t + 1 [m] のとき、項ごとに微分して
v(t) = dx/dt = 6t − 2 [m/s]
例えば t = 2 s では v = 10 m/s（正の向き）。

確認

1.50 s

解説

速さから行程を求める（積分）

速さ v(t) が分かれば、行程 s は速さを時間で積分して得られます。

s = ∫₀ᵗ v(t) dt　　(5.6)

これは v-t グラフと時間軸で囲まれた面積に等しい。

等速度運動（v = v₀ 一定）　式 5.6 を積分し、初期条件 t = 0 で s = s₀ とすれば
s = v₀t + s₀　　(5.7)
行程は時間に正比例する（s-t グラフは直線）。
※ 微分（速度を求める）と積分（行程を求める）は互いに逆の操作。
解説

§5.1.4

加速度（§5.1.4）

速度がどれだけ速く変わるか — それが加速度。

a = limΔt→0 Δv/Δt = dv/dt = d²r/dt²  (5.12)
|a| = dv/dt = d²s/dt²  (5.13)
  • 加速度は速度の時間微分、位置から見れば2階微分
  • 単位は m/s²（メートル毎秒毎秒）
  • 直線運動では a>0 で加速、a<0 で減速（＝逆向きに加速）

速度が「位置の変化率」だったのと同じ構図で、加速度は「速度の変化率」。微分をもう一段重ねた量です。

§5.1.4

等加速度運動の公式（§5.1.4）

加速度の大きさ a が一定（a = a₀）のとき、式5.14 dv/dt = a₀ を順次積分するだけで運動が決まります。
① 加速度を時間で積分（初期条件 v(0) = v₀）
v = a₀t + v₀（式5.15）
② さらに積分（初期条件 s(0) = s₀）
s = ½a₀t² + v₀t + s₀（式5.16）
③ 式5.15と式5.16（s₀=0）から t を消去
v² − v₀² = 2as（式5.17）
※ これら3式は等加速度運動の基本公式（教科書 p.82〜83）。
解説

等加速度運動の公式一覧

4つの公式を「含む量」と「含まない量」で整理すると、問題ごとに最適な式が選べます。
公式含む量含まない量
v = a₀t + v₀v, v₀, a₀, ts
s = ½a₀t² + v₀t + s₀s, s₀, v₀, a₀, tv
v² − v₀² = 2asv, v₀, a, st
s = (v₀ + v)/2 · ts, v₀, v, ta
問題で与えられた量と求めたい量を見て、含まない量で消去できる公式を選ぶ。

自由落下（free fall）

空気抵抗を無視したとき、重力のみによる運動。等加速度運動の特別な場合（a = g）です。鉛直上向きを正に取ると：
a = −g = −9.80 m/s²
自由落下の公式（上向き正）
v = v₀ − gt
y = y₀ + v₀t − ½gt²
v² − v₀² = −2g(y − y₀)
  • g = 9.80 m/s² は重力加速度の大きさ（教科書準拠）
  • 質量によらず同じ加速度（ガリレオの実験）
  • 符号に注意：正の向きを決めてから式を立てる
解説

自由落下の例（独自例題11.1）

問題
高さ 20 m のビルから物体を静かに落とす。地面に到達するまでの時間と着地時の速さを求めよ。
解法（落下方向を正に取る）：
① 初期条件と加速度
y₀ = 0,  v₀ = 0,  a = g = 9.80 m/s²
② 到達時間（v₀=0 なので）
y = ½gt² ⟹ t = √(2y/g) = √(2×20/9.80) ≈ 2.02 s
③ 着地時の速さ
v = gt = 9.80 × 2.02 ≈ 19.8 m/s

グラフの傾きと面積で読む

運動を表す3つのグラフから、傾き（微分）と面積（積分）でどんな量が読めるかを一望にする。
グラフ操作読み取れる量
x-t グラフ傾き速度 v
v-t グラフ傾き加速度 a
v-t グラフ面積行程・変位 Δx
a-t グラフ面積速度変化 Δv
直線 → 等速運動、放物線 → 等加速度運動、水平 → 静止（x-t グラフの場合）。傾き＝微分、面積＝積分。

例題5.1：列車の A–B 駅間（教科書 p.84）

問題
列車が A 駅を出発し B 駅へ向かう。等加速度で走行し2分後に一定速度 v₀=100 km/h に達する。出発後10分で等減速を始め、2分後に停止する。A–B 間の行程 s を求めよ。
v-t グラフは台形。その面積が行程 s。
加速 2分・等速 8分・減速 2分。
速度は km/min、時間は分で計算（→行程は km）。
s = ½(100/60)×2 + (100/60)×8 + ½(100/60)×2
s ≈ 16.7 km
加速:三角形＋等速:長方形＋減速:三角形。台形の面積＝行程。

まとめ

  • 運動学は力を問わず、運動の記述に集中する（§5.1）
  • 行程（経路全長・スカラー）と変位（ベクトル）、距離（変位の大きさ）を区別（§5.1.2）
  • 速度 v=dx/dt は位置の微分、速さはその大きさ（式5.4〜5.5）
  • 加速度 a=dv/dt=d²x/dt² は速度の微分（式5.12）
  • 等加速度の基本公式（式5.15〜5.17）と自由落下（a=g）
  • グラフの傾き（微分）と面積（積分）から物理量を読み取れる
